# Projet_Cloud_AccManager #

# Objectif #

Account Manager doit gérée les compte des utilisateurs et proposer des services par rapport à leurs compte. 
Chaque Compte (Account) est lié à un Client (Customers). 

# EndPoints #

1. Account
  - Get
    - / => Get All
    - /{id} => Find By Id
  - Post
    - / => Create
  - Put
    - /{id} => Update
